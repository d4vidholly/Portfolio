Peppermint OT

Brief: "Design and build an Occupational Therapy website. Support Coordinators will use it to review my suitability to their clients and contact me for appointments."

Roles: UX Researcher, Graphic / UI / UX Designer, Web Developer

  • Research: Users want a clear, easy-to-use webpage that outlines contact info, location and services. Through user interviews and developing personas, I found that users are frustrated by unclear websites and value building a relationship with the OT.
  • Design: Wireframing reduced 50% of the initial content to create a clearer message. Usability testing showed that a sticky CTA performed better than a nav bar link.
  • Outcomes: User feedback described the tone of the page as 'friendly' and 'fresh', supported by an effective visual hierarchy to emphasise key elements.

"David’s design research helped understand Support Coordinators, resulting in positive feedback about the brand and building strong professional connections." - Becky, Owner

HAiLO

Brief: How might we leverage road crash statistics and multi-agent AI-based web applications to enhance road safety and inform policy making?

Roles: UX Researcher, UI / UX Designer

Key Design Features:

  • Research: The team analysed crash data and external factors like wildlife, traffic signals, and holidays while consulting policymakers for insights. This combined approach ensured a thorough understanding of road safety challenges and solutions.
  • Design:HAiLO, an AI-powered map tool, was tested for seamless adoption by officials. Its user-friendly interface integrates diverse datasets, providing clear and actionable recommendations to address road safety challenges effectively.
  • Outcomes: HAiLO features an intuitive, informative interface that enhances data visualisation for officials. It supports campaigns and infrastructure changes with evidence-based insights, laying the foundation for scalable AI solutions in public safety.

"Congratulations to the team for their incredible work! One thing that stood out to all the judges was the strong adherence to the criteria and the simple yet highly informative user interface - Ashwin, Dept of Transport and Planning

The United Project

Brief: We don't make enough use of our current landing page. We need a redesign to meet our fundraising goals for 2025." Please Note: Redesign submitted as part of application

Roles: UX Researcher, UI / UX Designer

Key Design Features:

  • Research: Spot testing showed users thought landing page navigation was unclear and lacked obvious ways to get involved. Secondary research and competitor analysis showed branded donation pages can raise 6x more.
  • Design: Information Architecture audit helped reduce nav bar icons and informed including a Shop section on page. High-fidelity prototypes tested imagery and color to improve navigation on the page, strengthen brand identity and create a clearer donation message.
  • Outcomes: A/B Testing revealed users enjoyed the 'clean', responsive update. A clearer understanding of the mission and the concept of a donation goal connected users to get involved.

Empower Autism

Brief: "We need to redesign our current landing page to help users interact with our site."

Roles: UX Researcher, UI / UX Designer

Key Design Features:

  • Research: Interviews showed mobile users didn't trust the landing page which they felt was out of date. A competitive analysis was performed to identify best practices in landing page design. Card sorting process highlighted inconsistent Information Architecture.
  • Design: Wireframes visualised concepts to ensure logical flow. Clearer headings and increased CTAs to improve navigation. Used more appropriate imagery without text overlays to improve clarity and trust.
  • Outcomes: Users related more to the landing page and were more likely to interact with the site as a whole.
